# SOUP — Система Управления Проектами

**Автор:** Просвирнин Данил, 11-402

---

## Описание

**SOUP** — это современная система для планирования, отслеживания и управления задачами команд в рамках проектов. В системе реализованы роли пользователей (администратор, менеджер, разработчик, тестировщик), гибкая система прав доступа и инструменты для создания, назначения и мониторинга задач.

---

## Возможности

- **Многоуровневая архитектура:**  
  Три схемы (core, project, analytics)

- **Связи между сущностями:**  
  Один-к-одному, один-ко-многим, многие-ко-многим (10–15 таблиц)

- **Основные объекты:**  
  Пользователи, роли, проекты, задачи, итерации (спринты), комментарии, вложения, логи, агрегаты и активности

- **Оптимизация:**  
  Индексация по ключевым полям, отдельный tablespace для индексов, комбинированные индексы

- **Документирование:**  
  Комментарии ко всем объектам и атрибутам

- **Права доступа:**  
  Разные роли и ограничения через GRANT/REVOKE

---

## Технологии

- **DBMS:** PostgreSQL  
- **Создание и наполнение:** DBeaver  
- **Управление и запуск сервера:** pgAdmin  
- **Языки:** SQL, PL/pgSQL
